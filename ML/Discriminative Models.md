*one line - What separates these classes*
## Core purpose : 
**Predict** the **correct label** from **input data**.
*Rather than understanding how the data itself was generated.*
## They focus on :-
- classification
- decision boundaries
- predicting accuracy
- separating classes
## Core Idea : 
Suppose you have:
- Input features ***X***
- Labels ***Y***
A discriminative model learns : 
$$
P(Y∣X)
$$
or directly learns a function:
$$
f(X)→Y
$$
                  *Given this input, what is most likely the output?* 

## Intuition 
- Image Classifying:
	- cats vs dogs
	- spam vs noon spam
- It does NOT try to understand:
	- how cats are biologically formed?
	- how spam emails are generated?
- It asks, *WHAT SEPARATES THESE CLASSES?*
